    Dynamic NPC System(Unreal C++, Blueprints)[WIP]

    Virtual Reality games prioritize immersion and the playerâ€™s interactions with the world around them (Hand Physics, immersive object interaction, etc.) But they are behind in the field of Non player characters (NPCs) and interactions between the player and NPCs. The NPCs are often unresponsive 3D models.

    The project aim is to improve immersion in VR games by designing NPCs that react realistically to physical touch interactions.

    I am the sole developer of the project made in Unreal as part of my masters dissertation. The tasks and features I implemented include

  • VR player setup(VR hands) and NPC AI with Blackboard and Behaviour tree
  • A custom collision system that detects NPC body part and touch intensity
  • A finite state machine that uses the new collision system with different behaviour modes(Aggresive, friendly)
  • Physics based animation that moves the bones using the new collision system and ragdoll physics.
  • This dynamic system is designed to act as a plugin.
  • A demo level that features the usage of all the different features [WIP]
